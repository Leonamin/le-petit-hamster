# Roadmap

Build in **vertical slices**: one complete planet end-to-end before breadth.
Success is measured by *feeling* (warm, quiet, curious), not feature count.

## Milestone 0 — Tracer bullet ✅ (scaffolded)
- [x] Vite + React + TS + R3F project
- [x] Spherical-walk controller: hamster walks around a planet
- [x] Low follow-camera looking slightly upward
- [x] Mood pass: fog + bloom + vignette + twilight lighting
- [ ] Deployed to a live URL (Vercel/Netlify/Pages) — do this early!

## Milestone 1 — Rain Planet MVP (the one playable slice) ✅
- [x] Hand-authored diorama: lighthouse + landmarks (more polish later)
- [x] NPC: Mouse Lighthouse Keeper — approach to trigger dialogue
- [x] Dialogue UI (storybook text, minimal)
- [x] Optional friend: sleeping hamster — find & gently wake (optional!)
- [x] Ambient audio: procedural rain (Web Audio, no asset)
- [x] "Leave planet" fade/transition (loops back for now; real travel = M2)

## Milestone 2 — A second planet + travel
- [ ] Extract a Planet abstraction from RainPlanet
- [ ] Minimal planet-to-planet travel
- [ ] A second themed planet (Time / Memory / Friendship)

## Later / Backlog
- Save the planets you've visited
- Settings (audio, controls)
- Swap placeholder hamster for a CC0 rigged model with idle/walk animation
- Mobile / touch controls

## Cut from MVP (do NOT build yet)
- Planet-selection star map
- Multiple simultaneous NPCs
- Any progression / completion system (against the Golden Rule)
